# Reviewer Response Plan - Round 1

## Issues Addressed

### 1. Title/Framing Mismatch (CRITICAL - all 3 reviewers)
**Problem:** Title references "Decline of the Bank Teller" but data is NAICS 522110 (all commercial banking employment), not occupation-level teller data.
**Fix:** Reframe title and throughout to "banking employment" not "teller employment." Acknowledge the aggregation limitation prominently. Add employees-per-branch analysis to test consolidation mechanism.

### 2. Branch Pre-Trends (CRITICAL - all 3 reviewers)
**Problem:** Significant pre-trends in 2005-2006 for branches; F-test rejects parallel trends.
**Fix:** (a) Strengthen the discussion that employment null is unaffected by branch pre-trends. (b) Report a restricted sample (2007+) robustness check for branches. (c) Be more cautious in branch causal claims.

### 3. Exposure-Dynamism Correlation (HIGH - all 3 reviewers)
**Problem:** Positive placebo coefficients suggest Durbin exposure correlates with county growth.
**Fix:** (a) More explicitly discuss bias direction. (b) Note that positive bias means the true banking effect is likely more negative, but still small. (c) Add balance table or pre-period characteristic comparison by exposure quartile.

### 4. Reframe Paper Away From "Teller Decline"
**Fix:** Change title to remove "Decline of the Bank Teller." Reframe as banking employment broadly. Keep the Bessen ATM-teller hook but clarify we study the broader workforce.

### 5. Employees Per Branch Analysis
**Fix:** Construct and report employment-per-branch to directly test consolidation mechanism.
